## the defs element

While it’s acceptable to **use** a visible object, the better practice is to define reusable objects in the **defs** section.
You can think of **defs** as a storage place for graphical objects. Elements defined in **defs** aren’t rendered directly - they are only rendered when referred to by the **use** element.

<Blocks>{`<svg width="500" height="500" viewbox="0 0 500 500">

    <defs>
        <circle id="dot"
            cx="0" cy="0" r="100"
            fill="purple" />
    </defs>

    <use
        href="#dot"
        x="250" y="250"></use>

</svg>`}</Blocks>

<DefsExample />

A defs object can be a single shape element like a circle, rectangle, or a path, or it can be multiple elements that are grouped \<g\>.
