# Intro

## What you'll learn:

1. the **elements** and **syntax** of an SVG
1. how to create svgs using free drawing tools like **Figma**
1. how to create svg graphics dynamically using **JavaScript**
1. Styling SVGs using **presentation attributes** and **CSS**
1. How to make SVGs **interactive**
1. How to **animate** SVGs

## Prerequisites

### HTML

A basic understanding of html, and the fundamental elements of an html page.

### CSS

How to create and apply basic CSS styles

### Javascript

A fundamental understanding of variables, functions, and event listeners.

## Tools

1. A **Code Editor** like **Visual Studio Code** or **Atom**.
1. A drawing program capable of exporting files in the SVG format, like **Figma**
